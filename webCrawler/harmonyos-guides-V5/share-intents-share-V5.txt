共享联系人信息到分享推荐区
通过意图框架服务，目标应用可以将联系人信息共享到分享推荐区。参考：习惯推荐-接入方案。
该示例代码无法直接运行，需要申请意图框架白名单。参见：Intents Kit接入流程。
开发步骤
导入相关模块。import BuildProfile from 'BuildProfile';\nimport { util } from '@kit.ArkTS';\nimport { BusinessError } from '@kit.BasicServicesKit';\nimport { insightIntent } from '@kit.IntentsKit'; 目标应用构造联系人数据。const intent: insightIntent.InsightIntent = {\n  intentName: 'SendMessage', // 意图名\n  intentVersion: '1.0', // 意图版本\n  identifier: util.generateRandomUUID(), // 意图标识符\n  intentActionInfo: { // 意图执行信息\n    actionMode: 'EXECUTED', // 动作模式\n    executedTimeSlots: { // 实际发生时间段\n      executedStartTime: new Date().getTime(),\n      executedEndTime: new Date().getTime(),\n    }\n  },\n  intentEntityInfo: { // 意图实体信息\n    entityId: 'this-is-id', // 实体Id\n    entityName: 'Contact', // 实体名称\n    name: 'Nickname', // 联系人昵称\n    icon: 'data:image/png;base64,...', // 联系人头像\n    phoneNumbers: [], // 联系人电话号码\n    extras: {\n      shareParams: {\n        bundleName: BuildProfile.BUNDLE_NAME, // 应用包名\n        moduleName: 'entry', // 应用模块名 根据实际填写\n        abilityName: 'SampleContactAbility', // 应用ability名 根据实际填写\n        action: 'ohos.want.action.sendData', // 标识分享 不可修改\n      }\n    }\n  }\n}; 目标应用共享联系人数据。insightIntent.shareIntent(getContext(this), [intent]).then(() => {\n  console.info('shareIntent ok');\n}).catch((err: BusinessError) => {\n  console.error(`shareIntent failed. Code: ${err.code}. message: ${err.message}`);\n});
