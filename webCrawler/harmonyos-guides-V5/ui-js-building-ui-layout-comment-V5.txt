添加留言区域
留言框的功能为：用户输入留言后点击完成，留言区域即显示留言内容。用户点击右侧的删除按钮可删除当前留言内容并重新输入。
留言区域由div、text、input关联click事件实现。开发者可以使用input组件实现输入留言的部分，使用text组件实现留言完成部分，使用commentText的状态标记此时显示的组件（通过if属性控制）。在包含文本完成和删除的text组件中关联click事件，更新commentText状态和inputValue的内容。具体的实现示例如下：
<!-- xxx.hml -->\n<div class=\"container\">\n  <text class=\"comment-title\">Comment</text>\n  <div if=\"{{!commentText}}\">\n    <input class=\"comment\" value=\"{{inputValue}}\" onchange=\"updateValue()\"></input>\n    <text class=\"comment-key\" onclick=\"update\" focusable=\"true\">Done</text>\n  </div>\n  <div if=\"{{commentText}}\">\n    <text class=\"comment-text\" focusable=\"true\">{{inputValue}}</text>\n    <text class=\"comment-key\" onclick=\"update\" focusable=\"true\">Delete</text>\n  </div>\n</div>
/* xxx.css */\n.container {\n  margin-top: 24px;\n  background-color: #ffffff;\n}\n.comment-title {\n  font-size: 40px;\n  color: #1a1a1a;\n  font-weight: bold;\n  margin-top: 40px;\n  margin-bottom: 10px;\n}\n.comment {\n  width: 550px;\n  height: 100px;\n  background-color: lightgrey;\n}\n.comment-key {\n  width: 150px;\n  height: 100px;\n  margin-left: 20px;\n  font-size: 32px;\n  color: #1a1a1a;\n  font-weight: bold;\n}\n.comment-key:focus {\n  color: #007dff;\n}\n.comment-text {\n  width: 550px;\n  height: 100px;\n  text-align: left;\n  line-height: 35px;\n  font-size: 30px;\n  color: #000000;\n  border-bottom-color: #bcbcbc;\n  border-bottom-width: 0.5px;\n}
// xxx.js\nexport default {\n  data: {\n    inputValue: '',\n    commentText: false,\n  },\n  update() {\n    this.commentText = !this.commentText;\n  },\n  updateValue(e) {\n    this.inputValue = e.text;\n  },\n}
